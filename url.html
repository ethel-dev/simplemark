Sexydown 3: A sexy way to print Markdown.

Sexydown is a useful, completely public domain, free website that prints code beautifully minimalistic on paper. It can compile Markdown and Textile markup to HTML and print it with a variety of options, and it can print 118 languages in beautifully syntax-highlighted glory on paper. Better yet, you can create a custom URL using the URL generator that will check the options for you. Sexydown 3 has many new features that make it a fully-featured printing utility belt that you'll keep in your bookmarks for all eternity. If you're a nerd, you will be pleased to hear that Sexydown 3 is public domain on GitHub.
