    • Sexydown: A sexy way to print Markdown.

    Sexydown is a dead simple way to make regular Markdown (and Textile!) code look elegant and printable. All you have to do is copy and paste your raw Markdown, GitHub Flavored Markdown, or Textile code and Sexydown will make it elegant and give you a new, printable webpage. Anyone who uses Markdown for documents on a daily basis will find this website pretty helpful, or at least I hope they will!


    Tip: type S-E-X-Y to make it sexy, fast!
